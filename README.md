# vue3-demo-admin

## 描述

使用Vite Vue3 Ts 搭建的基础开发脚手架

❤️： 使用Vite作为前端构建工具
❤️： Vue3
❤️： Ts Tsx
❤️： VueRouter@4
❤️： Piana
❤️： 使用naive-ui 组件库 （按需引入）
❤️： 使用@vicons/ionicons5图标库 （按需引入）
❤️： 统一eslint代码规范
❤️： 统一git提交日志规范


## 安装

### yarn
yarn create vite-app-lv my-app

### npm 
1. npx create-vite-app-lv my-app
2. npm init vite-app-lv my-app
